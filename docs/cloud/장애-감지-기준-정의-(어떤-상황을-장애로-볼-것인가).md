## 1. 개요

본 문서는 서비스 운영 중 발생할 수 있는 장애 상황을 정의하고, 장애 여부를 판단하는 기준을 명시합니다.

---

## 2. 일반적인 고려 기준

| 기준 | 설명 |
|-----|------|
| 가용성 기준 | 서비스 응답 불가, 헬스체크 실패, 500번대 에러 발생 |
| 성능 기준 | 응답 시간 N초 초과, 타임아웃 발생 |
| 에러율 기준 | 전체 요청 중 에러 비율 N% 초과 |
| 리소스 기준 | CPU/메모리 N% 이상 지속, 디스크 풀, 커넥션 풀 고갈 |
| 품질 기준 | 정상 응답이지만 비즈니스 로직 오류 (빈 응답, 잘못된 데이터 등) |

---

## 3. 선정한 장애 기준

현재 MAU 500 규모에서는 **가용성 기준**만 적용합니다.

### 3.1 롤백으로 임시 해결 가능한 장애

배포가 원인일 경우 롤백을 통해 해결할 수 있는 장애입니다.

| 장애 상황 | 판단 기준 | 롤백 대상 |
|----------|----------|----------|
| 헬스체크 실패 | `/actuator/health` 응답 불능 또는 비정상 | moyeobab-api |
| 핵심 API 500 에러 | 핵심 API에서 500번대 에러 지속 | moyeobab-api |
| recommend 서비스 응답 불가 | recommend 서비스 코드 문제로 인한 장애 (배포 원인) | recommend |

**핵심 API 목록:**
- 카카오 OAuth 로그인
- 토큰 갱신
- 스와이프 투표 제출
- Top3 결과 조회
- 최종 식당 선택

### 3.2 롤백 대상이 아닌 장애

배포와 무관하거나 롤백으로 해결되지 않는 장애입니다.

| 장애 상황 | 원인 | 대응 방안 |
|----------|------|----------|
| DB 연결 실패 | DB 서버 문제 | DB 상태 확인, 연결 설정 점검 |
| Redis 연결 실패 | Redis 서버 문제 | Redis 상태 확인, 재시작 검토 |
| RunPod 응답 불가 | 외부 GPU 서비스 장애 | RunPod 상태 확인, Fallback 동작 여부 확인 |
| 외부 음식점 API 응답 불가 | 외부 API 제공자 장애 | API 제공자 상태 확인, 캐시 활용 검토 |
| 디스크 풀 | 저장 공간 부족 | 디스크 정리, 로그 삭제 |
| DB 커넥션 풀 고갈 | 커넥션 누수 또는 설정 문제 | 커넥션 누수 확인, 풀 사이즈 조정 |

---

## 4. 장애 판단 플로우

```
장애 알림 수신 또는 장애 인지
    ↓
장애 유형 확인
    ↓
┌─────────────────────────────────────┐
│ 헬스체크 실패 / API 500 에러        │ → 최근 배포 확인 → 배포 원인이면 롤백
│ recommend 서비스 장애               │ → 최근 배포 확인 → 배포 원인이면 롤백
├─────────────────────────────────────┤
│ DB/Redis 연결 실패                  │ → 해당 서비스 상태 확인 및 조치
│ RunPod/외부 API 장애                │ → 외부 상태 확인, Fallback 동작 확인
│ 디스크 풀 / 커넥션 풀 고갈           │ → 리소스 정리 및 설정 조정
└─────────────────────────────────────┘
```

---

## 5. 선정 이유

MAU 500 미만 규모에서 "서비스가 동작 안 하는 심각한 상황"을 우선 감지하기 위함입니다.

- 현재 규모에서는 가용성 기준만으로 충분하다고 판단
- 성능 기준, 에러율 기준, 품질 기준은 더 정교한 모니터링 인프라 필요
- 리소스 기준은 장애 발생 후 원인 조사 과정에서 확인

## 6. 향후 확장 (MAU 증가 시)

트래픽이 증가하면 Prometheus + Grafana 도입 후 다음 기준을 고려할 예정입니다.

| 기준 | 추가 항목 |
|-----|----------|
| 가용성 | recommend 헬스체크 |
| 성능 | 응답 지연 N초 초과 |
| 에러율 | 특정 API 에러율 N% 초과 |
| 품질 | 추천 결과 0건 (API 200이지만 빈 응답) |
